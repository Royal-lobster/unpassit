# unpassit

## 1.1.0

### Minor Changes

- Rename all references from unlockit to unpassit, add @clack/prompts for polished CLI UX with styled prompts, spinners, and structured log output

## 1.0.0

### Major Changes

- Initial release of unpassit - a CLI tool to remove passwords from protected files.

  Supports PDF, Office (doc/docx/xls/xlsx/ppt/pptx), ZIP, RAR, and 7z formats.
  Features auto-detection, batch mode, interactive password prompt, and env var support.
